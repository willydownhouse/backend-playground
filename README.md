# backend-playground

Small backend learning projects in one repo. Each folder is a standalone app with its own build and README.

## Projects

| Project | Stack | Description |
|---------|-------|-------------|
| [notification-system](./notification-system/) | Quarkus, Kotlin, Kafka, PostgreSQL | REST API with async notifications via Kafka |

## Getting started

Pick a project, `cd` into its folder, and follow that project's README.

```shell
cd notification-system
./mvnw quarkus:dev
```
